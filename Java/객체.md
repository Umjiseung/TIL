# 객체

## 객체란
* 객체: 의사나 행위가 미치는 대상, 문장 내에서 동사의 행위가 미치는 대상, 작용의 대상이 되는 쪽

* 자바에서 객체란 변수, 데이터 구조, 함수, 메서드가 될 수 있으며 식별자가 참조하는 메모리의 값이다.

* 객체는 변수, 함수, 데이터 구조의 조합이 될 수 있다.

* 객체는 다형성, 상속이 추가된 추상 데이터 유형이다.

## 객체, 오브젝트
* 객체 또는 오브젝트는 클래스에서 정의한 것을 토대로 메모리에 할당된 것으로, 프로그램에서 사용되는 데이터 또는 식별자에 의해 참조되는 공간을 의미하며, 변수, 자료구조, 함수, 메서드가 될 수 있다.
